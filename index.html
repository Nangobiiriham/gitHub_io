SELECT user_id, source_ip, COUNT(*) AS attempts
FROM login_attempts
WHERE status = 'failed'
  AND attempt_time > NOW() - INTERVAL '1 hour'
GROUP BY user_id, source_ip
HAVING COUNT(*) > 5

Security Automation Scripts

Developed Python and SQL scripts for automated security log analysis, brute force detection, and SIEM workflow automation — significantly reducing manual log review time in SOC environments.

Python SQL SOAR Log Analysis 
import re

def detect_brute_force(log_file):
    attempts = {}
    with open(log_file) as f:
        for line in f:
            if "failed login" in line:
                ip = re.search(r'\d+\.\d+\.\d+\.\d+', line)
                if ip:
                    attempts[ip.group()] = attempts.get(ip.group(), 0) + 1
    return {k: v for k, v in attempts.items() if v > 5}
